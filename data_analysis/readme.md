# 人民网"绘说现代化"抖音视频评论数据分析项目

## 项目简介

本项目用于分析人民网抖音账号"绘说现代化"合集中99个视频的评论数据，旨在为论文写作提供数据支撑。通过对视频评论、标题、互动数据的分析，提取公众对中国现代化建设相关话题的关注点和情感倾向。

---

## 目录结构

```text
data_analysis/

├── dock/               # MCP搜索结果文档目录
├── src/                # 源代码目录
└── readme.md           # 项目说明文档
```

## 研究目标

### 1. 高赞评论关键词分析

- 对格式化后的评论数据进行分析
- 提取高点赞评论中的关键词
- 输出结果保存至 `csvabstract/` 目录

### 2. 视频标题高频词分析

- 提取99个视频的标题
- 分析标题中的高频词汇
- 输出结果保存至 `csvabstract/` 目录

### 3. 高互动视频分析

- 筛选点赞量、分享量较高的视频
- 分析这些视频的评论关键词
- 探索高互动视频的共性特征

### 4. 相关政策文献检索（待实现） （看看有没有视频发布的时间，如果有的话就是用视频发布的时间进行查找那段时间的相关政策，时间范围可以放宽一点，如果没有就统计评论的大概的时间，使用评论的大概时间进行查询那一段时间的正常等，（查找的政策时间可以放宽一点））

- 根据视频主题搜索相关政策文件
- 使用MCP工具进行网络搜索
- 搜索结果保存至 `dock/` 目录

## 技术规范

### Python环境管理

- **包管理工具**: uv
- **虚拟环境目录**: `.venv/`
- **安装依赖**: `uv pip install <package>` 或 `uv add <package>`
- **执行脚本**: 使用 `.venv` 中的Python解释器

### 数据可视化库

根据图表类型选择最合适的库，可混合使用：

| 库名 | 适用场景 |
|------|----------|
| **Seaborn** | 统计图表、热力图、分布图 |
| **Plotly** | 交互式图表（静态导出） |
| **Altair** | 声明式可视化、复杂交互图 |
| **plotnine** | ggplot风格图表 |

### MCP搜索工具

| 工具名 | 用途 |
|--------|------|
| `brave_web_search` | 通用网络搜索（新闻、文章、政策文件等） |
| `brave_local_search` | 本地商业/地点搜索 |

## 工作流程

```text
1. 数据预处理
   output/ → 清洗格式化 → formatdata/

2. 数据分析
   formatdata/ → 关键词提取/统计分析 → csvabstract/

3. 数据可视化
   csvabstract/ → 图表生成 → figer/

4. 政策检索（可选）
   视频主题 → MCP搜索 → dock/
```

---

## 注意事项

1. 所有Python命令需通过uv管理的虚拟环境执行
2. 新增依赖使用 `uv add` 或 `uv pip install` 安装
3. 中间分析结果统一存放至 `csvabstract/` 目录
4. 可视化图表统一输出至 `figer/` 目录
5. 搜索结果文档保存至 `dock/` 目录并创建对应的MD文件
